# SQL操作数据库的专用工具

## 分类

  - 数据定义：DDL（Data Definition Language）
    - 库的创建删除、表的创建删除等
  - 数据操纵：DML（Data Manipulation Language）
    - 新增数据、删除数据、修改数据等
  - 数据控制：DCL（Data Control Language）
    - 新增用户、删除用户、密码修改、权限管理等
  - 数据查询：DQL（Data Query Language）
    - 基于需求查询和计算数据


## 特征
  - SQL大小写不敏感
  - SQL语句都以;结尾
  - 注释
    - 单行注释： -- 注释内容 （--后面一定要有个空格）
    - 单行注释： # 注释内容 （#后面可以不加空格， 推荐加上）
    - 多行注释：/* 注释内容 */
